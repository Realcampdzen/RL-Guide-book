# PRD: Чат-бот Путеводителя "Реальный Лагерь"

## 1. Обзор проекта

### Название
Чат-бот Путеводителя "Реальный Лагерь" - индивидуальный вожатый-эксперт по значкам

### Описание
Интеллектуальный чат-бот на базе OpenAI GPT-4o mini, который служит персональным вожатым-экспертом по системе значков проекта "Реальный Лагерь". Бот имеет доступ к полной базе данных значков (119 значков в 14 категориях) и может давать персонализированные советы, объяснения и креативные идеи для получения значков.

### Цель
Создать доступного 24/7 цифрового вожатого, который понимает педагогическую методику "Реального Лагеря" и помогает участникам эффективно развиваться через систему значков.

## 2. Целевая аудитория

### Основные пользователи
- Участники лагеря (дети и подростки 8-17 лет)
- Вожатые и педагоги
- Родители участников
- Организаторы лагеря

### Потребности пользователей
- Понимание системы значков и их целей
- Персонализированные рекомендации по развитию
- Креативные идеи для получения значков
- Объяснение педагогической философии
- Практические советы по достижению целей

## 3. Функциональные требования

### Основные функции
1. **Контекстное общение** - бот видит текущую категорию/значок пользователя
2. **Персонализированные советы** - рекомендации на основе интересов и уровня
3. **Объяснение методики** - понятные объяснения философии значков
4. **Креативные идеи** - практические предложения для получения значков
5. **Навигация по системе** - помощь в выборе подходящих значков

### Технические требования
- Интеграция с OpenAI GPT-4o mini API
- Доступ к базе данных ai-data/ (119 значков, 14 категорий)
- Система контекстного управления
- Обработка естественного языка
- Сохранение истории диалогов

## 4. Архитектура системы

### Компоненты
1. **Chat Bot Core** - основная логика бота
2. **Context Manager** - управление контекстом категорий/значков
3. **Data Loader** - загрузка данных из ai-data/
4. **OpenAI Integration** - интеграция с API
5. **Memory System** - сохранение контекста диалогов
6. **Response Generator** - генерация персонализированных ответов

### Технологический стек
- Python 3.8+
- OpenAI API (GPT-4o mini)
- JSON для данных значков
- Система промптов для личности бота
- Логирование и мониторинг

## 5. Пользовательский опыт

### Сценарии использования
1. **Изучение категории** - пользователь спрашивает о конкретной категории значков
2. **Выбор значка** - помощь в выборе подходящего значка для развития
3. **Получение совета** - практические рекомендации по достижению целей
4. **Понимание методики** - объяснение педагогической философии
5. **Креативное планирование** - генерация идей для проектов

### Интерфейс
- Текстовый чат-интерфейс
- Поддержка эмодзи и форматирования
- Контекстные подсказки
- Навигация по категориям

## 6. Личность бота

### Характер
- Дружелюбный и понимающий вожатый
- Эксперт по педагогике "Реального Лагеря"
- Креативный и вдохновляющий
- Терпеливый и поддерживающий

### Стиль общения
- Использует эмодзи и живой язык
- Объясняет сложные концепции простыми словами
- Дает практические советы
- Мотивирует и вдохновляет

## 7. Критерии успеха

### Метрики
- Время ответа < 3 секунд
- Точность рекомендаций > 85%
- Удовлетворенность пользователей > 90%
- Активное использование контекста > 80%

### Качественные показатели
- Понятность объяснений
- Практичность советов
- Креативность идей
- Соответствие педагогической методике

## 8. Ограничения и риски

### Технические ограничения
- Зависимость от OpenAI API
- Ограничения контекста токенов
- Необходимость качественных промптов

### Риски
- Неточные рекомендации
- Непонимание контекста
- Технические сбои API

## 9. План разработки

### Этап 1: Базовая архитектура (1-2 дня)

- Создание системы загрузки данных
- Базовая интеграция с OpenAI API

- Система управления категориями/значками
- Интеграция с базой данных ai-data/

### Этап 3: Личность и промпты (1-2 дня)

- Адаптация личности НейроВалюши
- Тестирование качества ответов

### Этап 4: Продвинутые функции (2-3 дня)
- Персонализация рекомендаций
- Креативная генерация идей
- Система памяти диалогов

### Этап 5: Тестирование и оптимизация (1-2 дня)
- Комплексное тестирование
- Оптимизация производительности
- Финальная настройка

## 10. Ресурсы

### Данные

- Системный промпт НейроВалюши
- OpenAI API ключ

### Инфраструктура
- Python окружение
- Система логирования
- Мониторинг производительности

## 11. Заключение

Чат-бот Путеводителя станет ключевым инструментом для персонализированного развития участников "Реального Лагеря", обеспечивая доступ к экспертной поддержке 24/7 и помогая каждому найти свой уникальный путь развития через систему значков.
